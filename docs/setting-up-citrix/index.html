Before setting up Citrix, you should have the following things ready:

  1. You should get an Email from ZF which contains a Ping QR Code, which looks like this:

  1. A temporary password. This will be either Mailed to you or you will get it via Slack.

  2. install the Citrix Workspace.

  3. Upon completion of the installation, you may encounter a specific page. At this point, please close the application. We will utilize a configuration file for further setup procedures at a later stage.

Register the Ping ID on your phone‚Äã

  1. Proceed to install the Ping ID application on your mobile device.
    1. Play Store: https://play.google.com/store/apps/details?id=prod.com.pingidentity.ping&pli=1
    2. App Store: https://apps.apple.com/us/app/pingid/id891247102
    3. Microsoft Store: https://www.pingidentity.com/en/resources/downloads/pingid.html
Info ‚ú®

Once installed, please utilize the QR code presented in Step 1, or manually input the code. This process will link your phone to the ZF identity management system and prompt for an OTP/Pin each time you log in.

Login Using Temporary Password‚Äã

  1. Proceed to access.zf.com
  1. Choose Ping ID Authentication / Sign in with Ping ID
  1. Please provide your ZF email or ZF number, and click SIGN ON

  1. After clicking "Sign On", a notification will be sent to your Ping ID app. Please proceed by approving this.

  2. After completing this step, you'll be directed to a new page where you're required to input your temporary password.

  3. Following this, you should be presented with the below page. We recommend using the light version at this time.

  1. You should then be able to view the remote desktops associated with your account.

    Info

    ‚ú® If you are not able to see any shared desktop, it means that your account is new and you may need to wait until it appears here

Setup Citrix app (Not using light mode)‚Äã

  1. Please navigate to your Citrix dashboard. Once there, locate and click on the settings icon in the top right corner. From the dropdown menu, proceed to 'Account Settings'.
